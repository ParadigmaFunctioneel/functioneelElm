## frajaweb

Het lesmateriaal is gemaakt op de server 'frajaweb', die gehost wordt door één van de ontwikkelaars.
+ [jupyterhub.frajaweb.com](https://jupyterhub.frajaweb.com/hub/login)
+ Hier inloggen kan met een github-account.
